# Saving source evidence

Use `save_script_source` only after a bounded region is selected. Supply the target
experiment and, when available, the initiating network evidence ID.

Reports should cite the saved script-source evidence ID and SHA-256. Distinguish:

- observed source text;
- inferred control/data flow;
- behavior confirmed by a browser experiment.

Do not publish credential literals, private source beyond the needed range, or claims
that depend only on bundle naming.

# Minimal capture

Start with the smallest experiment that can answer the question.

- State one objective.
- Use an empty flow or one snapshot for a baseline.
- Allow zero primary matches when discovering traffic.
- Enable only required capture channels.
- Add narrow network selectors after endpoint evidence exists.
- Avoid user-visible mutations unless the question requires them.
- Use one experiment per independent intervention.

Escalate from page/console/network summaries to exact evidence only after the baseline
shows what is missing.

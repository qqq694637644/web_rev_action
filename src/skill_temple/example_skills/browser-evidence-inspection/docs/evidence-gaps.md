# Evidence gaps

Record a gap when evidence is missing, truncated, stale, partial, or belongs to a
different request/session/experiment.

For each gap write:

1. the unanswered question;
2. what was inspected;
3. why the available evidence is insufficient;
4. the minimum next evidence needed;
5. whether a new consequential experiment is required.

Never fill a gap from product memory, historical behavior, or an unrelated capture.

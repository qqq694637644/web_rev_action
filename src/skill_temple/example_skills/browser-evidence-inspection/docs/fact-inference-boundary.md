# Fact and inference boundary

Use three explicit layers:

- **Observation:** a value directly present in a manifest, bounded response, or artifact.
- **Comparison:** a deterministic difference between named observations.
- **Inference:** an interpretation supported by observations but not directly recorded.

Do not turn a script text match into proof of request construction. Do not turn one
successful replay into proof that a field is universally optional. State scope,
account/session context, and experiment IDs with conclusions.
